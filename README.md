# Simon Says Game 🎮

A fun and interactive **Simon Says** memory game built using **HTML**, **CSS**, and **JavaScript**. The game challenges players to remember and repeat an increasingly long sequence of colors.

## 🚀 Features

* Interactive gameplay with colorful buttons
* Random sequence generation
* Increasing difficulty with each level
* User input validation
* Game over screen with score display
* Restart and play again functionality

## 🛠️ Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling, animations, and responsive design
* **JavaScript (ES6)** – Game logic and user interactions

## 🎯 How to Play

1. Press any key to start the game.
2. Watch the highlighted color carefully.
3. Click the buttons in the same order as shown.
4. Each correct sequence advances you to the next level.
5. If you click the wrong button, the game ends and your score is displayed.
6. Press any key to restart and try again.

## 💻 Installation
